Selectors

  • Pseudo-element Selectors

    The pseudo-element selectors in CSS are used to apply specific styles to certain parts of an element's content, such as the first letter, first line, or a part of the text. They are represented by the double colon (::) notation and are combined with selectors to target specific portions of an element's content.

Selectors

  • Pseudo-Classes

    A pseudo-class selector is HTML code that changes the appearance of certain parts of a webpage when the user interacts with it i.e. by hovering over or clicking on a link

Properties

  • Opacity

    The value of opacity is a number between 0.0 (0%) and 1.0 (100%). To use opacity you add an 'a' to your colour value. E.g. color: rgba(100,100,100,0.5).

  • Border-radius

    The border-radius CSS property rounds the corners of an element's outer border edge. You can set a single radius to make circular corners, or two radii to make elliptical corners.

Layouts

  • Absolute Positioning

    An absolutely positioned element is an element whose computed position value is absolute or fixed .

  • Flex

    An efficient and dynamic way to lay out, align and distribute space among items in a container. The container can alter its items’ width/height (and order) to best fill the space.

  • Display inline

    The display property specifies the display behavior (the type of rendering box) of an element. Element displayed as an inline element similar to span does not start on a new line. The element only takes up the required width.
