Selectors

  • Pseudo-Class Selectors

    Pseudo Class selectors can be appended to Element Selectors and enable elements to be targeted based on an aspect of their state. The syntax for a pseudo class selector is to append an element selector with a single colon followed by the specified state that the selector should target. They’re a good way of adding “dynamic” styling to elements without resorting to using JavaScript.

  • Attribute Selectors

    The CSS attribute selector matches elements based on the presence or the value of a given attribute.

  • Pseudo-Element Selectors

    A Pseudo-Element Selector is used to style specific parts of an element. For example to style the first letter or line of an element and to insert content before, or after, the content of an element

  • ID

    A unique identifier used in HTML that you can style. You must not use the same ID elsewhere in your document. In the internal/external area of styling the prefix is a hash (#) followed by your ID name.

  • Element Selector

    A CSS selector is the first part of a CSS Rule. It is a pattern of elements and other terms that tell the browser which HTML elements should be selected to have the CSS property values inside the rule applied to them.

Properties

  • Box-shadow

    The box-shadow CSS property adds shadow effects around an element's frame. You can set multiple effects separated by commas. A box shadow is described by X and Y offsets relative to the element, blur and spread radius, and color.

  • Padding

    Padding is used to create space around an element's content, inside the borders. There are properties for setting the padding for each side of an element: top, right, bottom, and left.

  • Background Image

    The background-image property enables the display of an image in the background of an element.

  • Margins

    The Margin element of CSS, defines the outside portion of the box model.

    It creates a space around an element, which wraps around the boarder.

    Margin sizes are set from Top, Left, Bottom, Right - just think of a clock!

    They can be defined by longhand e.g "margin-top" or just by defining the size by number.

Layouts

  • Flex

    CSS flexible box layout is a way to layout, align & distribute space among items in a container. Especially when the size of those elements is unknown or dynamic.

  • Absolute Positioning

    Positions elements on the page relative to the next parent element with relative or absolute positioning.

  • Position: Fixed

    An element positioned with "fixed" will always stay in the same place relative to the viewport, even when scrolling.

  • z-index

    The z-index property in CSS controls the vertical stacking order of elements that overlap. As in, which one appears as if it is physically closer to you. z-index only affects elements that have a position value other than static (the default).

  • Display

    The display CSS property sets whether an element is treated as a block or inline element and the layout used for its children, such as flow layout, grid or flex.

  • Grids

    Grid Container - The element
    Grid Item - The children
    Grid Line - The dividing lines
    Grid Cell - The space between two adjacent rows.
    Grid Track - The space between two adjacent grid lines.
