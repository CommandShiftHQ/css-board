CSS Board

Selectors

  • class

    Class selectors The CSS class selector matches elements based on the contents of their class attribute.

  • ID

    The id selector selects an element based on the value of its id attribute. There should only be ONE element with a given id in a document.

  • Attribute Selectors

    The attribute selector matches elements based on the presence of a given attribute.
    This can be further specified by adding a value that the attribute contains.
    Different tags can be added to the selector to further specifiy matches.

Properties

  • Border-Radius

    The border-radius CSS property rounds the corners of an element's outer border edge. You can set a single radius to make circular corners, or two radii to make elliptical corners.

  • Padding

    CSS padding is used to create space around an elements content.
    Using the 'Inspect' option on the dropdown menu will help you see where your padding is applicable

  • Box-Shadow

    This CSS property adds a shadow around an element's frame. The colour and size of the effect can be changed. For example each post-it you can see here has a black box-shadow.

  • Margin

    The margin property creates space around a chosen element.

Layouts

  • Absolute Positioning

    An absolutely positioned element no longer exists in the normal document flow. Instead, it sits on its own layer separate from everything else. This is very useful: it means that we can create isolated UI features that don't interfere with the layout of other elements on the page. For example, popup information boxes and control menus etc.
