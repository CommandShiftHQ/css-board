Selectors

  • Pseudo-elements

    A CSS pseudo-element is a keyword added to a selector that lets you style a specific part of the selected element(s). For example, p::first-line can be used to change the font of the first line of a paragraph.

  • Attribute Selectors

    Attributes are things inside elements that give us extra info. E.g. attribute="value". Some of the most common examples are'href' 'src' and 'alt'.
    To target these attributes, we use attribute selectors,they look like this [attribute=value]

  • ID

    Ids, on the CSS side, are represented by a hash (#), followed by the id name. You can target an element using its ID name and apply CSS rueles to it.

Properties

  • box-shadow

    The box-shadow property attaches one or more shadows to an element.

  • Margin

    This property can be used to set a margin on all four sides of an element. Margins create extra space around an element, unlike padding, which creates extra space within an element.

  • Border

    The border property lets you define the width, style and colour of your border

  • Padding

    The padding property sets the padding area on all four sides of an element at once. The padding property may be specified using one, two, three, or four values. Each value is a length or a percentage.
