CSS Board

Selectors

  • ID

    Ids, on the CSS side, are represented by a hash (#), followed by the id name. You can target an element using its ID name and apply CSS rueles to it.

Properties

  • box-shadow

    The box-shadow property attaches one or more shadows to an element.

Layouts

    Grid

    Grid is a 2D grid-based system where you can explicitly define the number of rows and columns of your layout, or allow the browser to implicitly define the number and size of the rows and columns.

    "display: grid;" will turn a parent container into a grid container.
