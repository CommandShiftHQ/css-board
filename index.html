Selectors

  • Psuedo Element Selectors

    I am a key word added to a selector that lets me be styled specifically

  • Attribute Selectors

    Within HTML, an attribute selector is a way to select elements and style them based on their attribute values.

  • Pseudo-Class Selectors

    A class used to define a different or special state of an element. They can be used for hovering behaviours, affecting links or styling a link when it gets focus.

  • Combinator Selectors

    Combinator selectors allow you to combine multiple selectors to target specific elements. Examples include descendant, child, sibling and adjacent selectors

Properties

  • Float

    Float specifies how an element should float within its container. It will be positioned either left or right of its containing element or other floated elements.It accepts the following values:left, right, none and inherit

Properties

  • Opacity


    The opacity property specifies the opacity/transparency of an element.

  • Border-Radius

    It defines the radius of your element's corners. It can have 1 to 4 values. Starting with the first value is the;top-left, top-right,bottom right then bottom-left. This property can round corners of an object. e.g. border-radius: 15px 10px 3px 40px;

    • Margin

      The margin property is used to add a transparant area around an element. This will push other content away from the box and can be useful to arrange things on the page.

Layouts

  • Absolute positioning

    When an element is set with absolute positioning, this element position is relative to its parent

    • Display: Inline

      Elements are displayed one after another, with no line break between them. Height and width properties have no effect on inline elements. Examples include <span> and <a>.
